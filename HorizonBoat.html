Predicting selling price with Python¶

Cleaning, analysing, visualizing and modeling data using Linear Regression¶

By Alexandre Kocev

Horizon Boat company is building and selling private boats in Europe. Willing to launch a new motor boat, they are asking us to analyse the European boat market. As a Data Analyst, we gathered data including a list of boats with their characteristics and price. Our goal is to define the features which have the greatest impact on price, and build a model to predict the price using Linear Regression.

The original data was created by Artem and collected from here. It was then reorganized by myself for the purpose of this case study.

We have enough observations for each category. We continue our analysis with a statistical method : ANOVA.

ANOVA or Analysis of Variance is used to determine differences between the means of several grouped variables. It returns two parameters :

  • F-test score calculates the difference between the mean for each group. Larger difference between means suggest better correlation.
  • P-value shows how statistically significant is our correlation between variables.

STEP 4 : Predict Price with Multiple Linear Regression¶

MLR, or Multiple Linear Regression is used to calculate a dependant variable (i.e. Price) based on multiple independent variables (i.e. predictors selected in STEP 3 : Length, Width, Number of Cabins, Boat Type, Fuel Type).

(1) Pre process data with Dummy variables¶

Our categorical variables (i.e. Number of Cabins, Boat Type, Fuel Type) need to be transformed to Dummy variables in order to perform MLR. Dummy variables -or indicator variables- are used to convert each distinct attribute in categorical variables into zero or one. Zero represents the absence of the attribute and one represents the presence.

Doing so allows us to model our linear relationship.

(4) Write down our Linear Regression¶

Our model shows good accuracy, so we can now use it for making Price predictions.

The equation of Multiple Linear Regression is given as :
Y = a + b1.X1 + b2.X2 + bn.Xn

Where 'Y' is the dependant variable (i.e. Price) and each 'X' represents one predictor (i.e. Length, Width, Number of Cabins, Boat Type, Fuel Type).
'a' is the intercept which is the theoretical value of the dependent variable when predictors are set to zero.
'b' is the coefficient assigned for each predictor.
